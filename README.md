<h1>🍡 Ate Fes Kakanin</h1>

<p>
A modern, responsive <strong>React + Vite</strong> project showcasing a fictional Filipino
kakanin business. The website highlights traditional Filipino delicacies through a clean,
contemporary interface while demonstrating modern front-end development practices.
</p>

<h2>✨ Features</h2>

<ul>
  <li>Responsive single-page website</li>
  <li>Modern React component architecture</li>
  <li>Built with Vite for fast development</li>
  <li>Smooth scrolling navigation</li>
  <li>Scroll reveal animations</li>
  <li>Interactive product showcase</li>
  <li>Responsive gallery section</li>
  <li>About and story section</li>
  <li>Contact information with social links</li>
  <li>Clean and reusable CSS architecture</li>
  <li>Accessible semantic HTML</li>
</ul>
